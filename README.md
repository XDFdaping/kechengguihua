# 课时规划生成器 · v8

新东方大坪校区销售专用工具，根据本地知识切片 PDF 库 + 学生层级（潜能/中等/优等）一键生成课时规划。

## 在线地址
https://XDFdaping.github.io/kechengguihua/

## v8 升级要点
- **学生层级选择器**：年级下拉后新增「潜能生 / 中等生 / 优等生」三选一，生成课次时直接读取切片中对应群体的"建议课次"数字
- **章节化切片区**：按章节分组展示，每张卡片显示「知识点 + 内容」摘要
- **PDF 导出**：jsPDF + autoTable，嵌新东方品牌色（深绿 #1A5C38 + 金黄 #E8A020）
- **数据重抽**：原 3055 条噪声 → 542 条真实知识切片（数学 275 + 英语 267），字段：chapter / points / content / hours_low / mid / high

## 使用方式
1. 填学生姓名
2. 选年级 + 学生层级
3. 选辅导科目（多选）
4. 展开章节，勾选需要的知识切片
5. 选排课方式（暑假连排 / 常规周排 / 自定义日期）
6. 填总课时数 + 备注
7. 点「生成课时规划」→ 复制 / 下载 Excel / 下载 PDF

## 配套脚本
- `extract_pdfs.py` — 扫 G 盘 PDF 知识切片库，输出 `knowledge_slices_data.js`
- `build_v8.py` — 把 v3 `index.html` 改造成 v8（注入学生层级、PDF 导出等）
